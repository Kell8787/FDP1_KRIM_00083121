2. ¿Que es una matriz?

La matriz es un arreglo de datos, pero no estan ordenados en linea si no que estan ordenados
binimensionalmente osea en filas y columnas. 

Sintaxis de una Matriz:

tipo nombre [tamaño 1][Tamaño 2]... 
-Por ejemplo:

#include <stdio.h>
int main(){

	int Numeros[3][4];

return 0;
}

donde int: es el tipo
Numeros: Es el nombre de la varibale
[3]: es el tamaño
[4]: es el tamaño
